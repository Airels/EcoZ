# EcoZ

University Web Project using NodeJS

Improve yourself to make the world better !

## Objects structures :
- Question :
    - id
    - title (text of question)
    - creator (username of author)
    - creationDate (using timestamp)
    - listIDAnswers (Splitted with ',')
    - idCorrectAnswer
- Answer :
    - id
    - content
- User :
    - id
    - username
    - password
    - mail
    - hereSince (timestamp)
    - totalScore
    - isAdmin
    - isPremium
    - description
